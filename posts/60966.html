variables in JAVA

Variable ——coffee beans of JAVA program.

The variables in JAVA⬇

flowchart TD
    B["variables in JAVA"]
    B-->C["Member variable"]
    B-->D("Local variable");
    C-->E("Class variable/Static variable")
    C-->F("Instance variable")

Memeber varible

成员变量顾名思义就是类中直接下属的变量，在类中的使用没有局限性，这里根据变量生存周期和访问方式的不同又分为类变量（Class variable）和实例变量（Instance variable）

实例变量的生存周期与实例的产生消亡一致；类变量与类共存亡

  • 两者的定义都是直接在class(Person)内部的
  • 实例对象的使用必须需要一个object
  • 静态变量的调用往往通过类名，同一个类中的静态变量的值统一
  • 成员变量是由默认值0的，而局部变量没有

Local variable

局部变量声明在方法、构造方法或者语句块中；当方法执行完后，变量将会1被销毁。

  • 局部变量只在声明它的方法、构造方法或者语句块中可见；
  • 局部变量是在栈上分配的
