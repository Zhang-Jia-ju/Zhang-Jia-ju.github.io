super & this in Java

所有对象的父类都是 Object 类，这就意味着，所有的对象实例化的时候都需要先调用 Object 的无参构造方法;

而连接对象和 Object 至少需要一条道路，而 this 和 super 就是这个最重要的桥梁;

"this"

1.属性名

用来区分局部变量和成员变量

输出：↓

20 18

this.age输出成员变量age的值，age输出局部变量age的值。（两个变量虽同名，但地址空间是完全不同的）

2.方法名

用于在方法中调用同类的其他方法

输出：↓

method1 is running method2 is running

method2 中 加入 this.method1(),就在方法2中调用了方法1的代码

3.构造方法

与2类似，在构造函数中调用不同参数的构造函数

但是这里注意一下，this语句必须放在该构造方法的第一行

"super"

首先先写上一个子父类

1.访问父类构造方法

在父类中的定义两个构造方法

子类 Student 继承了 Person 类，使用 super 语句来定义 Student 类的构造方法

PS：编译器会自动在子类构造方法的第一句加上super();来调用父类的无参构造方法，必须写在子类构造方法的第一句，也可以省略不写。

2. 访问父类成员

当子类的成员变量或方法与父类同名时，可以使用 super 关键字来访问

这里和this关键字的使用类似

输出：↓

学生年龄：12

3.调用父类成员方法

当父类和子类都具有相同的方法名时，可以使用 super 关键字访问父类的方法

输出：↓

This is student class This is person class
