Class in Python

类是模板，而实例则是根据类创建的对象。

一、创建和使用类

使用类几乎可以模拟任何东西，在类中，主要是由属性和方法构成的，属性是一个状态名词（比如小狗的颜色、品种、年龄和体重等等） 而方法是一个行为（比如小狗吃、跳跃、奔跑、叫等等）。

四、内置类属性

  • dict : 类的属性（包含一个字典，由类的数据属性组成）
  • doc :类的文档字符串（输出文件开头字符串的内容）
  • name: 类名
  • module: 类定义所在的模块（类的全名是'main.className'，如果类位于一个导入模块mymod中，那么className.__module__ 等于 mymod）
  • bases : 类的所有父类构成元素（包含了一个由所有父类组成的元组）
